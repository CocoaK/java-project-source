叮咚美信IOS版APP操作指南

叮咚美信APP六大功能：监控、小区公告、通话记录、管理中心和设置。

 

 

 

 

 

开锁

回家时，无需繁琐的输入楼道门口机密码，用手机点击APP即可完成一键开锁操作。

监控

只要是手机或pad连接了网络，您随时随地都能反向查看门口机摄像头前的情况，完成远程反向监察，并可同步开门操作。

物业通知

物业费用信息、物业公共通知直接发送到您手机，在也不用每次贴到公告栏苦找物业通知，未读信息将会及时提醒您，不漏接任何消息。同时支持用户报修、小区内部的贴吧等功能。

通话记录

每位用户的相关记录可追溯100条，在特别情况下还可反向追查开门记录，保障有效时间内的可追溯性。

管理中心

可随时一键呼叫管理中心，报警、求助、报修等一键完成操作

设置

彰显人性化的四大功能，让您的使用得更加得心应手。

 

 

“设置”中的四个功能彰显人性化，让您使用得更加得心应手

 

 

 

每个用户都可以独享自己的二维码，而此二维码是由128位加密和4K的校验码设计，从根本上保证了安全性，加倍保护您的隐私。

在此设置开锁密码之后，方便您在未携带手机的情况下，可以使用个人设置的密码开门。

APP操作指南

 

 

 

参数设置：

 

SipAcc、SipPsw、SipDomain此三项为系统设置，请不要自行修改。

BandWidth为带宽设置，可以根据自己带宽条件手动设置宽带，在保证视频清晰度的前提下节省流量。默认为512，如果您需要更流畅，更省流量，但对画质要求不高，建议您可以设置为256.
